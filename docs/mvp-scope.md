# MVP Scope

## In Scope for MVP

- Supabase authentication
- Organization and main branch setup
- Staff profiles and roles
- Dashboard with live totals
- Product and service catalog
- Customers
- POS checkout
- Invoice history
- Payments and partial payments
- Customer balance view
- Repairs list and status updates
- Expenses
- Basic daily closing
- Audit log

## Not in First Foundation

- Full POS checkout implementation
- PDF rendering
- Thermal printer integration
- Returns/refunds
- Offline sync
- Advanced supplier purchase orders
- Barcode hardware integration

## Success Criteria

- The cloud app is separate from the desktop app.
- Authenticated users only see their organization data.
- The cashier can create real invoices without local desktop dependencies.
- Reports match the desktop app’s core financial meaning.
